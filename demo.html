🔧 Setting Up Real AI Recognition

To use actual food recognition:

  1. Visit Teachable Machine
  2. Create an Image Project
  3. Add classes for different foods (candy, apple, pizza, etc.)
  4. Upload training images for each food type
  5. Train your model
  6. Export as TensorFlow.js
  7. Update the MODEL_URL in script.js with your model URL

💡 Tips for Better Results

  • Use good lighting when scanning food
  • Hold the camera steady during capture
  • Ensure the food item fills most of the scanning frame
  • Try different angles if recognition isn't accurate

📞 Troubleshooting

Camera not working?

  • Make sure you're accessing via https:// or localhost
  • Check browser permissions for camera access
  • Try refreshing the page

App not loading?

  • Ensure JavaScript is enabled in your browser
  • Check browser console for errors (F12)
  • Try a different browser
